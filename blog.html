1.What is the DOM/ What is the Purpose of DOM?

Answer:DOM stands for Document Object Model. It is a programming interface that allows us to create, change, or remove elements from the document. We can also add events to these elements to make our page more dynamic.
=> One of the main reasons for developing the DOM is to maintain a standard programming interface for HTML and XML documents, which is language-independent. Setting and maintaining this standard enables programmers to create or navigate these types of documents and modify their elements predictably, using any type of language or development environment.

2.How to select element from DOM?

Answer:
1.getElementById - search element by element_id.
2.getElementsByTagName - search element by tag name.
3.etElementsByClassName - search element by class name.
4.getElementsByName - search element by name attribute.

3.What is Event Bubble?

Answer: Event Bubble is a concept in the DOM . It happens when an element receives an event, and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element.

4.what is the callback function?

Answer: Any function that is passed as an argument to another function so that it can be executed in that other function is called as a callback function.
